# GUI dan Collection

GUI (Graphical User Interface) ini digunakan untuk memvisualisasikan dan mengelola data yang disimpan dalam beberapa jenis collection di Java, seperti **List**, **Set**, dan **Map**. Melalui tombol-tombol yang tersedia, data dapat dimasukkan, disimpan ke collection, diambil kembali, dan dikelola (ditambah, diubah, atau dihapus).

## 1. List

List merupakan collection yang dapat menyimpan data secara berurutan. Pada List, nilai yang sama atau duplikat diperbolehkan.

Pada contoh berikut, data **Sepatu, Sandal, Kaos kaki, Kaos, Celana, Celana** dapat muncul lebih dari satu kali.

## 2. Set

Set merupakan collection yang tidak memperbolehkan adanya nilai duplikat.

Pada contoh berikut, jumlah data yang dimasukkan ke dalam List adalah 6 data. Namun, ketika data tersebut dimasukkan ke dalam Set, jumlah data yang tersimpan menjadi 5 karena nilai **Celana** hanya disimpan satu kali.

## 3. Map

Map merupakan collection yang menyimpan data dalam bentuk pasangan **key** dan **value**.

Setiap elemen pada Map memiliki kunci yang digunakan untuk mengakses nilainya. Pada Map, key tidak boleh duplikat. Jika terdapat key yang sama, maka value sebelumnya dapat digantikan oleh value yang baru.
